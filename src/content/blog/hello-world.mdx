# Test

## Markdown testing

MDX combines the best of markdown and JSX:

1. **Simple syntax** for content creation
2. **Component embedding** for interactive elements
3. **Code highlighting** out of the box
4. **Dynamic content** possibilities

```bash
npx create-next-app@latest my-blog
cd my-blog
npm install next-mdx-remote gray-matter
```
